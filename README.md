# Hlupa's DotFiles
My DotFiles to makeup clear Debian install.

## Requirments
Install Git:

    apt install git

## Installation
RUN AS ROOT

Commands list:

    git clone git@github.com:715MrDydosinka/dotfiles.git /usr/share/dotfiles
    
    cd /usr/share/dotfiles
    
    ./setup_root.sh
    
And just follow the steps in script;
    
Done.
