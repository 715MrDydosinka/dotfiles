# Hlupa's DotFiles
My DotFiles to makeup clear Debian install.

## Requirments
Install Git:

    `apt install git`

## Installation
Commands list:

    `git clone git@github.com:715MrDydosinka/dotfiles.git ~/.local/dotfiles`
    
    `cd ~/.local/dotfiles`
    
    `sudo ./setup_root.sh`
    
And just follow the steps in script;
    
Done.
